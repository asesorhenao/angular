Variables de referencia de plantilla
Nos ayuda a porder hacer referencia a un elemento de una plantilla. Este elemento típicamente será un elemento del DOM o un componente. Ejemplo: input type="file" #uploader

Ejemplo: Botón para carga de archivo

ViewChild: Referenciando un componente desde una clase
Podemos hacer referencia a elementos de la plantilla desde la clase del componente, se pueden utilizar dos formas:

Ejemplo 1: Utilizando una variable de referencia de plantilla. Ejecutar una función desde el componente hijo al componente padre. Seria
@ViewChild('hijo'), y la referencia en la etiquita del componente
app-variables-referencia-hijo #hijo.
Para este caso se utiliza el tipo del objeto

Escribe algo: {{ mensajeDeError }}
ViewChildren: Referenciando varios componentes desde una clase
Con ViewChildren podemos agarrar una conjunto de instancias de un componente de una plantilla
